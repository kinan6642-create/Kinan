==================================================
طريقة الحصول على ملف Setup.exe بدون جهاز ويندوز
==================================================

هذي الطريقة تخلي أجهزة ويندوز حقيقية تابعة لـ GitHub (مجانية بالكامل) تبني لك
ملف التثبيت تلقائياً وتعطيك رابط تنزيل جاهز، خلال ٥-٧ دقائق تقريباً.

--------------------------------------------------
الخطوة ١: إنشاء حساب GitHub (إذا ما عندك)
--------------------------------------------------
1. روح لموقع: https://github.com/signup
2. سجّل حساب مجاني (يحتاج بريد إلكتروني فقط)

--------------------------------------------------
الخطوة ٢: إنشاء مستودع (Repository) جديد
--------------------------------------------------
1. بعد تسجيل الدخول، اضغط على زر "+" أعلى الصفحة، ثم "New repository"
2. اكتب أي اسم (مثلاً: cashier-pos)
3. اختر "Private" (خاص، حتى لا يشوفه أحد غيرك)
4. اضغط "Create repository"

--------------------------------------------------
الخطوة ٣: رفع الملفات
--------------------------------------------------
1. بصفحة المستودع الجديد، اضغط "uploading an existing file"
   (أو من تبويب Add file > Upload files)
2. افتح المجلد اللي فك ضغطه من الملف اللي أرسلته لك
3. حدد كل الملفات والمجلدات بداخله (Ctrl+A) واسحبها لصفحة GitHub دفعة وحدة
   مهم جداً: تأكد إن مجلد ".github" مرفوع أيضاً ضمن الملفات (هو المسؤول عن البناء التلقائي)
4. انزل تحت واضغط "Commit changes"

--------------------------------------------------
الخطوة ٤: انتظار البناء التلقائي
--------------------------------------------------
1. اضغط على تبويب "Actions" أعلى صفحة المستودع
2. بتلاقي عملية بناء شغالة تلقائياً (دائرة صفراء تدور)
3. انتظر لين تتحول لعلامة صح خضراء (تاخذ ٥-٧ دقائق تقريباً)

--------------------------------------------------
الخطوة ٥: تنزيل ملف Setup.exe
--------------------------------------------------
1. اضغط على عملية البناء (اللي عليها علامة الصح الخضراء)
2. انزل تحت لقسم "Artifacts"
3. اضغط على "تثبيت-نظام-الكاشير-Setup" لتنزيله (بيجي كملف مضغوط zip)
4. فك الضغط، وبتلاقي بداخله ملفين:
   - تثبيت-نظام-الكاشير-Setup-ia32.exe  → لأجهزة ويندوز 32-bit
   - تثبيت-نظام-الكاشير-Setup-x64.exe   → لأجهزة ويندوز 64-bit (أغلب الأجهزة الحديثة)

اختر الملف المناسب لجهاز المستخدم. إذا ما تعرف نوع الجهاز، جرب نسخة x64
أولاً (هي الأكثر شيوعاً بالأجهزة الحديثة)، وإذا ما اشتغلت استخدم ia32.

كل ملف يعمل على: ويندوز 7 و10 و11 (حسب المعمارية المطابقة) ويضيف اختصار
تلقائياً على سطح المكتب وقائمة ابدأ عند التثبيت.
